Project Overview – NetCommand

NetCommand is a web-based network configuration tool designed to streamline the setup of enterprise switches for students and IT professionals. Built using React.js and Flask, the system allows users to configure VLANs, assign ports, enable port security, and access a simulated terminal interface that mimics real Cisco CLI behavior.

The goal of this project is to provide a clean, accessible frontend with modular form inputs, backed by a Python-driven Flask server that interprets and logs network configuration commands. This approach helps users learn and manage network setups without requiring direct access to physical hardware.
